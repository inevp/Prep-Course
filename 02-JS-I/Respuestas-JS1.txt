VARIABLES: Una variable es un espacio reservado a la que puede almacenar datos dentro del nombre asignado. Por ejemplo puedo guardar el número 11 en una variable llamada Nro y luego usar esa variable para hacer operaciones matemáticas. 
STRING: son tipo de datos alfanuméricos.
FUNCIONES: Las funciones son códigos ejecutables que se almacenan bajo un nombre. Se les asigna un NOMBRE DE FUNCION y ARGUMENTOS. Los argumentos son variables que nos permiten pasar datos para realizar los cálculos dentro de la función.
La sintaxis de una función es:
FUNCTION NOMBREDELAFUNCION (argumento1,argumento2,…) { }
RETURN: especifica un valor que devuelve la función y finaliza la función. Se usa para mostrar valores fuera de la función y terminar la ejecución de la función.
IF: Su traducción es “Si”. Es una instrucción de CONTROL DE FLUJO. Se utiliza para verificar el cumplimento de condiciones. Permite tomar decisiones: Se evalúa de CONDICION, si la condición se cumple, se ejecuta un código. 
La sintaxis es: 
IF (CONDICION) { }
La condición usa operadores de comparación (>,<, >=,<=,==,==!) Para evaluar la condición. Si la condición se cumple, las instrucciones entre corchete se ejecutan.
VALORES BOOLEANOS: Los valores booleanos representan valores de verdad. Se usa TRUE para el valor “Verdadero” y FALSE para el valor “Falso”.
